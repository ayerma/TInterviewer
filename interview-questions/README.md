# Interview Questions Repository

This directory contains curated interview questions and answers organized by technology and topic.

## Directory Structure

```
interview-questions/
├── java/
│   └── core-java-fundamentals-and-oop-principles.md
└── README.md (this file)
```

## Organization

- **Technology Folders** (e.g., `java/`, `spring/`, `python/`)
  - Each technology has its own folder
  - Markdown files are named using kebab-case matching the topic

## File Naming Convention

- Use kebab-case: `topic-name-and-description.md`
- Be descriptive but concise
- Match the topic name from the source

## Content Structure

Each markdown file includes:
1. **Metadata** - Topic name, question count, source, generation date
2. **Table of Contents** - Quick navigation to all questions
3. **Questions and Answers** - Clearly formatted Q&A pairs
4. **Code Examples** - Where applicable, with proper syntax highlighting
5. **Summary** - Overview and study recommendations

## Contributing

When adding new interview questions:
1. Create appropriate technology folder if it doesn't exist
2. Follow the established markdown format
3. Include table of contents for 5+ questions
4. Add code examples where helpful
5. Include metadata and source information
