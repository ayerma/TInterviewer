Issue: IN-23
Summary: Implement breadcrumb navigation component

Description:
Description:

Build a breadcrumb component that displays the current navigation path (Space > Topic > Question). Each breadcrumb segment should be clickable and navigate to that level. The breadcrumb should update automatically based on the current route. Display at the top of the main content area above the question title.



Technical Notes:

Use @solidjs/router useLocation to derive breadcrumb segments from the current path. Use <For> to render breadcrumb items with separators (e.g., ' > ' or chevron icon). Each segment should be a clickable link using <A> component from @solidjs/router. Style with TailwindCSS with hover states. Handle edge case where user is at Space or Topic level (not Question level).



Type: component

Story Points: 2



Acceptance Criteria:

- Breadcrumb displays current path as Space > Topic > Question

- Each breadcrumb segment is clickable and navigates to that level

- Breadcrumb updates automatically when route changes

- Last segment (current page) is not clickable or styled differently

- Breadcrumb handles partial paths (e.g., only Space or Space > Topic)

- Responsive truncation on narrow screens
