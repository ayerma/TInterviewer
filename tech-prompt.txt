# Current Task

Issue: IN-20
Summary: Implement collapsible hierarchical navigation menu component

Description:
Description:

Build the left sidebar navigation menu that displays the Space > Topics > Questions hierarchy. The menu must support expand/collapse functionality at each level. Clicking on a Space or Topic shows its children as a list of hyperlinks. Clicking on a Question navigates to that question's content in the main area. The menu should maintain its expanded/collapsed state during navigation. Implement using SolidJS reactive patterns with proper state management for UI state (expanded nodes).



Technical Notes:

Use SolidJS createSignal or createStore to manage expanded/collapsed state for each node. Use <For> component to render lists recursively. Implement toggle icons (chevron/arrow) that rotate on expand/collapse. Style with TailwindCSS. Menu should be fixed position and scrollable independently. Consider using a recursive component pattern for nested levels. Highlight the currently active item based on route.



Type: component

Story Points: 5



Acceptance Criteria:

- Menu renders all Spaces, Topics, and Questions from spaces-index.json

- Clicking Space/Topic expands/collapses to show children as hyperlinks

- Clicking Question navigates to question detail view

- Expand/collapse state persists during navigation within the same session

- Currently selected item is visually highlighted

- Menu is collapsible (can hide/show entire left sidebar)

- Responsive behavior for mobile (menu overlay or full-width)
